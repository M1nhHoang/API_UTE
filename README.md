# API_UTE

Đại Học Sư Phạm Kĩ Thuật - Đà Nẵng - 2023

## Installation

1. Clone the project.
2. Edit the account and password in `config.json`.
3. If registering for a course, update the courses in `dsMonHoc.txt`.
4. Run `RunThisFile.py`.
